# design/cut_v2

已按真实 UI 结构重切。

- `01总览` 到 `05故障推演`：大屏展示页，切为顶部栏、左侧面板、主舞台、右侧面板、底部时间轴/导航/操作区。
- `06底图编辑`：机房平面编排页，单独按资源库、画布、属性配置、图层/校验/发布区域切。
- `07设备编辑`：机柜设备编排页，单独按设备库、U位画布、属性/端口/规则、底部发布区域切。
- `08拓扑编辑`：拓扑关系编排页，单独按节点库、拓扑画布、对象属性/规则/建议、底部版本发布区域切。

`manifest.json` 记录每个裁片的像素坐标和归一化坐标，可直接用于代码还原布局参考。

注意：这些裁片来自生成式位图，文字和图标是烘焙内容。开发时应作为视觉参考和布局定位依据，最终 UI 建议用原生组件重建。
