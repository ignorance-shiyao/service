
# **一、总览页（首页）**
  
## **组件1：业务运行总览**
- **形式**：指标卡（4~6个）
- **内容**：
  
  - 站点数 / 在线站点数
  - 隧道数 / 量子隧道数
  - 当前告警数
  - 业务可用率
  
- **客户价值**：
  
  - 一眼判断“业务是否正常”
  
- **数据来源**：
  
  - SD-WAN控制器工作台指标（设备数/隧道数/带宽等）
- **缺口**：
  
  - “量子隧道数”需从量子隧道标识接口获取（厂家已有UI，但API需确认）
  
------
## **组件2：量子能力分层（新增核心）**
- **形式**：三段式卡片（配置→启用→生效）
- **内容**：
  
  - 已配置量子服务器数
  - 已启用量子站点数
  - 实际量子隧道数
  
- **客户价值**：
  
  - 避免“误以为全网已量子加密”
  
- **数据来源**：
  
  - 量子服务器配置（控制器配置章节）
- 站点量子选择
- **缺口**：
  
  - “生效隧道数”需要隧道+量子状态联合
  
------
## **组件3：区域站点分布**
- **形式**：地图气泡
- **内容**：
  
  - 各区域站点数 / 异常数
  
- **客户价值**：
  
  - 判断区域性风险
  
- **数据来源**：
  
  - 地图模式（网元地理位置）
------
## **组件4：业务健康趋势**
- **形式**：折线图
- **内容**：
  
  - 可用率趋势
  - 告警趋势
  
- **客户价值**：
  
  - 看业务是否在恶化
  
- **数据来源**：
  
  - 性能管理（隧道/接口统计）
------
## **组件5：TOP风险站点**
- **形式**：滚动表
- **内容**：
  
  - 站点名 + 健康评分 + 当前问题
  
- **客户价值**：
  
  - 快速定位问题点
  
- **数据来源**：
  
  - 告警 + 性能（需融合）
  
- **缺口**：
  
  - 无现成“健康评分”，需中台计算
  
------
# **二、量子安全专题页（核心差异页）**
## **组件1：量子服务链路状态（关键）**
- **形式**：链路流程图
- **内容**：
```
控制器 → 量子服务器 → 设备 → 隧道
```
- 每段状态
- **客户价值**：
  
  - 定位量子不生效原因
  
- **数据来源**：
  
  - 量子服务器配置 + 设备连接状态
- **缺口**：
  
  - 无统一链路状态接口 → 需拼接
  
------
## **组件2：密钥健康度**
- **形式**：仪表盘 + 分布图
- **内容**：
  
  - 密钥剩余量
  - 即将耗尽数
  
- **客户价值**：
  
  - 防止业务中断
  
- **数据来源**：
  
  - 量子密钥信息
------
## **组件3：密钥更新趋势**
- **形式**：折线
- **内容**：
  
  - 更新成功/失败次数
  
- **客户价值**：
  
  - 判断量子服务稳定性
  
- **数据来源**：
  
  - 密钥更新情况
------
## **组件4：量子隧道分布**
- **形式**：环图
- **内容**：
  
  - 量子 vs 非量子
  
- **客户价值**：
  
  - 看覆盖范围
  
- **数据来源**：
  
  - 隧道信息（隧道编排）
------
## **组件5：加密类型分布**
- **形式**：环图
- **内容**：
  
  - 量子+国密 / 商密
  
- **客户价值**：
  
  - 安全等级感知
  
- **数据来源**：
  
  - 设备量子能力
------
## **组件6：量子异常列表**
- **形式**：滚动表
- **内容**：
  
  - 密钥不足 / 连接失败
  
- **客户价值**：
  
  - 提前预警
  
- **数据来源**：
  
  - 告警日志
------
# **三、拓扑页（结构解释能力）**

- **形式**：拓扑图
- **内容**：
  
  - 节点 + 链路 + 量子标识
  
- **客户价值**：
  
  - 理解网络结构
  
- **数据来源**：
  
  - 拓扑监控
------
## **组件2：量子路径追踪（新增）**
- **形式**：路径高亮
- **内容**：
  
  - 是否全路径量子
  
- **客户价值**：
  
  - 判断业务是否真正安全
  
- **数据来源**：
  
  - 拓扑 + 隧道 + 量子标识
  
- **缺口**：
  
  - 无直接接口 → 必须中台计算
  
------
## **组件3：链路状态分布**
- **形式**：环图
- **内容**：
  
  - 正常/异常
  
- **数据来源**：
  
  - 网元状态
------
## **组件4：拓扑事件流**
- **形式**：滚动时间轴
- **内容**：

- **数据来源**：
  
  - 隧道编排日志
