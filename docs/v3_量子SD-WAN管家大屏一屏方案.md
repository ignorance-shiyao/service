# 政企业务智慧运维管家 - 量子+SD-WAN管家 一屏方案

# 一、栅格布局

```
┌───────────┬───────────┬───────────┬───────────┐
│           │           │           │           │
│    A1     │    A2     │    A3     │    A4     │  第1行：量子安全
│  业务概况  │  量子密钥  │  量子隧道   │  加密算法  │
│           │  健康度    │  覆盖率    │  合规分布  │
│           │           │           │           │
├───────────┼───────────┼───────────┼───────────┤
│           │           │           │           │
│    B1     │    B2     │    B3     │    B4     │  第2行
│           │  量子服务  │  业务性能   │  TOP排行   │
│  安徽地图   │  器状态   │ [Tab切换]  │ [Tab切换]  │
│ [地图/拓扑] │          │           │            │
│  纵跨2行   ├───────────┼───────────┼───────────┤
│           │           │           │           │
│           │    C2     │    C3     │    C4     │  第3行：告警
│           │  告警概况  │  告警分析  │  告警趋势   │
│           │  [Tab切换] │ [Tab切换] │ [Tab切换]  │
│           │           │           │           │
└───────────┴───────────┴───────────┴───────────┘
```

实际模块数：12格 - 1格(地图多占1格) = **11个模块位**，其中6个支持Tab切换，总信息承载量 ≈ 22个视图。

---

## 二、各模块详细规格

---

### ▎第1行：量子安全(客户最关心，首行独占)

---

#### A1 · 业务概况

| 项 | 内容 |
|---|------|
| **指标** | ① 站点总数(个) ② CPE设备数(个) ③ 隧道总数(条) ④ 业务可用率(%) |
| **形式** | 上半区：4个数字指标卡(带小图标，2×2排列)；下半区：业务可用率进度条(沿用专线管家99.82%样式) |
| **客户价值** | 一眼掌握组网规模与整体健康度 |
| **数据来源** | SD-WAN手册1.1工作台(设备总数/隧道数)；2.1网元监控(在线/离线数)；可用率口径待厂家对齐 |

---

#### A2 · 量子密钥健康度

| 项 | 内容 |
|---|------|
| **指标** | ① 充注密钥总量(KB) ② 剩余密钥(KB) ③ 剩余比例(%) ④ 最近更新时间 |
| **形式** | 中心环形进度图(剩余/总量比例，<20%变红)；环外四角放辅助数字 |
| **客户价值** | 量子密钥是"弹药"——客户要知道弹药够不够、何时补充 |
| **数据来源** | SDWAN量子配置4.2.1(查看秘钥信息截图：剩余30624Byte/总量30720Byte/有效期20290122174837)；量子展示P1(同一截图) |

---

#### A3 · 量子隧道覆盖率

| 项 | 内容 |
|---|------|
| **指标** | ① 量子加密隧道数 ② 普通IPSec隧道数 ③ 量子覆盖率(%) |
| **形式** | 环形图(量子紫 vs 普通灰)，中间大数字显示覆盖率百分比 |
| **客户价值** | 直观回答"我的网络有多大比例受量子保护" |
| **数据来源** | 量子展示P1(隧道列表"量子状态"列：UP=量子加密，"--"=无量子)；SDWAN量子配置4.1(量子隧道叠加国密/商密表) |

---

#### A4 · 加密算法合规分布

| 项 | 内容 |
|---|------|
| **指标** | ① 国密(SM3+SM4)隧道数与占比 ② 国际(SHA256+AES256)隧道数与占比 |
| **形式** | 水平堆叠条(国密红色调、国际蓝色调) + 两端百分比标签 |
| **客户价值** | 等保2.0/密评场景下证明"用了国密算法"，合规可截图 |
| **数据来源** | SDWAN量子配置4.1(隧道表Hash算法列SM3/SHA256，加密算法列SM4/AES256) |

---

### ▎第2行 + 第3行左列：地图/拓扑(纵跨2格)

---

#### B1(纵跨第2-3行) · 安徽站点地图 / 网络拓扑

格内Tab切换，2个视图：

**Tab① 站点地图(默认)**

| 项 | 内容 |
|---|------|
| **指标** | 安徽省地图上各站点点位，按状态三色：正常绿、告警橙、离线红；量子站点加紫色外圈高亮；气泡内显示该地市站点数 |
| **形式** | 安徽省地图 + 气泡点位(沿用专线管家-农业银行截图的安徽地图+各地市气泡数字样式) |
| **悬停交互** | 鼠标悬停弹出：站点名/类型(CPE/POP)/设备型号/隧道数/量子状态 |
| **客户价值** | 看到组网地理覆盖，哪个地市站点多、哪个地市有异常 |
| **数据来源** | SD-WAN手册2.3地图模式(网元地理位置)；3.2站点管理(省/市/区/详细地址)；量子展示P3(物理拓扑地图) |

**Tab② 网络拓扑**

| 项 | 内容 |
|---|------|
| **指标** | POP(汇聚)↔CPE(接入)的Overlay拓扑图；实线=隧道，紫色/蓝色=量子隧道，灰色=普通隧道；节点颜色=在线/离线/告警 |
| **形式** | 力导向拓扑图(沿用量子展示P3用户拓扑截图：蓝色字体标注量子隧道) |
| **点击交互** | 点击节点弹出设备详情，点击链路弹出隧道SLA(时延/丢包/抖动) |
| **客户价值** | 看到Overlay组网结构，哪些路径有量子保护 |
| **数据来源** | SD-WAN手册2.4拓扑监控；量子展示P2-P3(物理拓扑+用户拓扑) |

---

### ▎第2行：中间两格

---

#### B2 · 量子服务器连接状态

| 项 | 内容 |
|---|------|
| **指标** | 量子服务器(QSS)列表：名称、IP:端口、连接状态(已连接/断连)、keepAlive、协议版本、QSSP应用标识 |
| **形式** | 状态列表卡片(2~3行)，在线=绿色圆点，断连=红色圆点+闪烁 |
| **客户价值** | 量子服务是独立的第三方服务——如果QSS断连，量子隧道退化为普通加密，客户需要第一时间知道 |
| **数据来源** | SDWAN量子配置4.2.1(连接状态表：abc/qkapp/v1.1/keepAlive=0/111.39.251.120/6001/已连接/WTSDK)；量子展示P1(同一截图) |

---

#### B3 · 业务性能(格内Tab切换，4个视图)

**Tab① 时延(默认)**

| 项 | 内容 |
|---|------|

| **数据来源** | SD-WAN手册第8章性能管理(接口/隧道统计) |

**Tab② 丢包**

| 项 | 内容 |
|---|------|
| **指标** | 按地市/站点分组的丢包率走势 |
| **形式** | 多系列折线图(沿用专线管家"站点抖动"截图样式) |
| **数据来源** | 同上 |

**Tab③ 抖动**

| 项 | 内容 |
|---|------|
| **指标** | 按地市/站点分组的抖动走势 |
| **形式** | 多系列折线图 |
| **数据来源** | 同上 |

**Tab④ 隧道流量**

| 项 | 内容 |
|---|------|
| **指标** | 选定隧道的入向/出向流速(Mbps)趋势 |
| **形式** | 双折线图(入向蓝/出向绿) |
| **数据来源** | 量子展示P2(基于隧道的信息统计：入向流速/出向流速折线图) |

#### B4 · TOP排行(格内Tab切换，3个视图)

**Tab① 带宽TOP5(默认)**

|---|------|

| **形式** | 横向条形图(沿用SD-WAN手册1.1工作台"设备带宽TOP5"截图样式) |
| **数据来源** | SD-WAN手册1.1工作台 |

**Tab② 设备性能TOP5**

| 项 | 内容 |
|---|------|
| **指标** | CPU/内存/温度异常TOP5设备 |
| **形式** | 横向条形图 |
| **数据来源** | SD-WAN手册2.1网元监控(CPU 4%/内存56%/温度27℃) |

**Tab③ 密钥剩余最低TOP5**

| 项 | 内容 |
|---|------|
| **指标** | 密钥剩余比例最低的5台CPE设备(名称/剩余量/总量/比例) |
| **形式** | 横向条形图(剩余<20%标红) |
| **客户价值** | 提前发现即将密钥枯竭的设备 |

### ▎第3行：告警与运维(地图右侧3格)

---

#### C2 · 告警概况(格内Tab切换，2个视图)

**Tab① 告警等级分布(默认)**

| 项 | 内容 |
|---|------|
| **指标** | 一级(严重)/二级(主要)/三级(次要)/四级(警告)告警各数量与占比 |
| **形式** | 环形图(沿用算网管家-政务云"告警等级"截图：四色分区，中间大数字=告警总数) |
| **数据来源** | SD-WAN手册5.1活动告警(告警级别字段) |

**Tab② 实时告警列表**

| 项 | 内容 |
|---|------|
| **指标** | 最新5~8条未确认告警(级别标色/网元名/原因/时间) |
| **形式** | 滚动表格，严重告警行高亮红色 |
| **数据来源** | SD-WAN手册5.1活动告警 |

---

#### C3 · 告警分析(格内Tab切换，2个视图)

**Tab① 告警原因分析(默认)**

| 项 | 内容 |
|---|------|
| **指标** | 网络侧/设备侧/客户侧/量子服务侧各占比 |
| **形式** | 环形图(沿用5G管家-安凯汽车"预警原因分析"截图样式) |
| **客户价值** | 故障定界——问题出在网络、设备、客户还是量子服务 |
| **数据来源** | SD-WAN手册5.1(告警原因字段)；量子服务侧分类需厂家提供量子告警码字典做映射 |

**Tab② 告警高频站点TOP5**

| 项 | 内容 |
|---|------|
| **指标** | 告警最多的5个站点(站点名/告警数/平均故障时长) |
| **形式** | 横向条形图(沿用专线管家"故障高频区县"截图样式) |
| **数据来源** | SD-WAN手册5.1+5.2(按网元汇总) |

---

#### C4 · 告警趋势(格内Tab切换，2个视图)

**Tab① 历史告警趋势(默认)**

| 项 | 内容 |
|---|------|
| **指标** | 近30天每日告警数走势 |
| **形式** | 柱状图(沿用算网管家-政务云"历史告警统计"截图样式) |
| **数据来源** | SD-WAN手册5.2历史告警 |

**Tab② 密钥更新统计**

| 项 | 内容 |
|---|------|
| **指标** | 入向/出向密钥更新次数(成功)、入向/出向更新失败次数(响应失败+校验失败+其他)，按天统计 |
| **形式** | 双折线图(成功折线+失败柱状叠加) |
| **客户价值** | 密钥更新是量子加密持续工作的心跳——失败突增意味着安全降级风险 |
| **数据来源** | SDWAN量子配置4.2.1(查看量子密钥更新情况：入向更新3次/出向更新3次/出向响应失败7次等完整字段) |

## 三、格内Tab切换汇总

| 格位 | Tab数 | 各Tab标签 | 默认Tab |
|------|-------|---------|--------|
| B1 地图(纵跨) | 2 | 站点地图 · 网络拓扑 | 站点地图 |
| B3 业务性能 | 4 | 时延 · 丢包 · 抖动 · 隧道流量 | 时延 |

## 五、视觉设计要点

| 要素 | 规格 |
|------|------|
| 底色 | 深蓝(沿用现有管家) |
| 量子专属色 | 紫色(#7c3aed)用于：量子隧道线条、地图量子站点外圈、A3覆盖率环形图 |
| 国密色 | 红色调(#dc2626)用于A4国密条 |
| 国际色 | 蓝色调(#2563eb)用于A4国际条 |
| 告警四色 | 红/橙/黄/蓝(严重→警告) |
| Tab切换样式 | 格子内顶部小字Tab，下划线激活态，不占用太多纵向空间 |
| 地图样式 | 安徽省轮廓填充深蓝底，各地市边界浅色描边，气泡点位(沿用专线管家截图) |
